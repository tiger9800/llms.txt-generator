# Automated llms.txt Generator

A Python-first web application that crawls a public website and generates a
standards-compliant `llms.txt` file.

## Repository Layout

```text
app/       FastHTML entry points and route handlers
models/    Shared typed domain models
services/  Crawl, extraction, prioritization, and generation logic
tests/     Automated tests
utils/     Focused reusable helpers
docs/      Project documentation
```
